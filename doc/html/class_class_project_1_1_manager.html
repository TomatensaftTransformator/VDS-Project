if-then-else computation for 3 input BDD's

ite(i, t, e) = or(and(i, t), and(not(i), e)) used to implement boolean-functions e.g. and, or, not, nand on BDD nodes

Parameters
iBDD_ID of the root of the funnction that represents the "if"
tBDD_ID of the root of the funnction that represents the "then"
eBDD_ID of the root of the funnction that represents the "else"
