virtual void ClassProject::ReachableInterface::setDelta ( const std::vector< BDD_ID > &  transitionFunctions)
pure virtual

Each state variable has a transition function.

The transition function specifies the value of the state after the transition. The transition function are composed of only state variables. For example: s0' = s0 XOR s1 The next state is defined as XOR of the current values of the state bit s0 and s1

Parameters
transitionFunctionsprovide a transition function exactly for each state bit

◆ setInitState()

virtual void ClassProject::ReachableInterface::setInitState ( const std::vector< bool > &  stateVector)
pure virtual

Each state machine has an inital state.

The inital state is provided as a vector. The vector has to have an entry for each state bit. If the entry is "true" the state bit is high, otherwhise negated. E.g. initial state not(s0) and not(s1) is transformed into {false,false}.

Parameters
stateVectorprovide the assignemtn for each state bit
